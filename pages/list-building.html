Bulk upload multiple listings

Add many buildings/units at once. Download the template, fill one row per unit, then upload. To list a building with several units, use one row per unit and give them the same building_ref (and repeat the building columns on each row) — rows that share a building_ref are grouped into a single building. amenities and certifications are semicolon-separated (e.g. 24/7 access;Security;Parking). For photos & floorplans, paste shareable links (Google Drive, Dropbox, SharePoint, OneDrive, or direct image/PDF URLs) in the photo_url_* / floorplan_url_* columns — no file uploads needed in bulk. Every row is validated server-side and stored as pending review; an admin then approves the batch in one click.
